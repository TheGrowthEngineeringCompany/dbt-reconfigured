# dbt reconfigured

Utilities created by [reconfigured](https://reconfigured.io). These macros try to be simple and composable.

## Installation

Add

```yaml
packages:
  - git: https://github.com/TheGrowthEngineeringCompany/dbt-reconfigured
    revision: v0.1.2
```

To `packages.yml` and run `dbt deps`

## Tests

```bash
./run_tests.sh
```
